Discord Python dili ile yapılmış discord botu
yenilikler getilinerek kullanışlı bir hale getirilebilir,
aklıma pekbirşey gelmediğinden bu halde bıraktım.


What you need:
Python 3.14




Setup:
1)Open Visual studio code
-file-Open Folder and select folder

2) Create .env
And type this code: DISCORD_TOKEN=MTQ1NzQ1Nz...
(MTQ1NzQ1Nz... your token)

3) Open bot.py in the terminal and type the following in order:

first:
pip3 install discord.py python-dotenv

second: If necessary (you can tell if this is necessary or unnecessary by looking at the terminal).
python.exe -m pip install --upgrade pip

4) Edit the config.json file according to your server's specifications (IMPORTANT!).
last 5) While inside bot.py, press F5 and then run it.
